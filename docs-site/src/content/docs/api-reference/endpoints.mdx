## POST /deploy

Trigger a manual deployment for a project.

### Request

**Headers:**
```
Cookie: connect.sid=<session-id>
Content-Type: application/json
```

**Body:**
```json
{
  "projectId": "<uuid>"
}
```

### Response

**200 OK:**
```json
{
  "success": true,
  "deployId": "<uuid>"
}
```

**401 Unauthorized:**
```json
{
  "success": false,
  "error": "Authentication required"
}
```

**404 Not Found:**
```json
{
  "success": false,
  "error": "Project not found"
}
```

## POST /webhook/:projectId

Receive a GitHub push webhook. No authentication required — uses HMAC-SHA256 verification.

### Request

**Headers:**
```
Content-Type: application/json
X-Hub-Signature-256: sha256=<hex-digest>
X-GitHub-Event: push
```

**Body:** GitHub's standard push event payload.

### Response

**200 OK:**
```
Deployment triggered
```
(Plain text response)

**401 Unauthorized:**
```
Unauthorized
```
(Plain text — HMAC signature mismatch)

**400 Bad Request:**
```
No body
```
(Plain text — missing or invalid payload)

## POST /email/test

Send a test email to verify SMTP configuration.

### Request

**Headers:**
```
Cookie: connect.sid=<session-id>
Content-Type: application/json
```

### Response

**200 OK:**
```json
{
  "success": true,
  "message": "Test email sent"
}
```

**400 Bad Request:**
```json
{
  "success": false,
  "error": "SMTP not configured"
}
```
