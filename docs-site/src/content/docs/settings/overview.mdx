## Settings Pages

VPS Deployer provides two settings pages accessible from the **Settings** section in the sidebar:

| Page | Path | Description |
|------|------|-------------|
| **Email Settings** | `/settings/email` | Configure SMTP for deployment notifications |
| **Path Settings** | `/settings/path` | Configure system PATH for command execution |

## User Details

Your account information is displayed at the top of the main Settings page:

- **Username** — your login name
- **Email** — your registered email address
- **Created At** — when your account was created
- **Last Logged In** — timestamp of your last login
